The planning board shows load by line and by day.

## What this page is for

It shows:

- what is planned on each line,
- on which day productions are anchored,
- follow-up tasks without making them consume line capacity.

## Frequent misunderstanding

Tasks shown later do not mean the manufacturing line is still occupied on those later dates.
