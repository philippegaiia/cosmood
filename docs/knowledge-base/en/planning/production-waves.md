A wave groups several productions under one planning and procurement view.

## What this page is for

It helps you:

- group batches,
- track consolidated needs,
- link supplier orders,
- replan several productions together.

## Frequent question

### Why use a wave instead of several standalone productions?

Because a wave gives one consolidated view of needs, procurement commitments, and allocatable stock.
