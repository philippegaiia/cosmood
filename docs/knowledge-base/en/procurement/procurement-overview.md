The procurement overview consolidates needs by ingredient and production context.

## What this page is for

It helps users understand the difference between:

- available stock,
- reserved stock,
- stock kept as reserve,
- stock allocatable to a wave,
- already ordered,
- already received,
- still to secure.

## Frequent misunderstanding

Planning coverage is not the same as real execution readiness.
