Ingredients are the basis for formulas, procurement, and stock.

## What this page is for

It defines:

- ingredient identity,
- category,
- base unit,
- code,
- internally manufactured flag,
- packaging flag,
- minimum stock alerts.

## Common mistake

Choosing the wrong base unit creates confusion later in purchasing, stock, and production quantities.
