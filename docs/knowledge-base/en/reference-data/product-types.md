The product type is the operational framework of a product.

## What this page is for

It defines:

- batch sizing mode,
- default batch size,
- expected units,
- estimated waste,
- shared QC template,
- allowed production lines,
- default line.

## Frequent question

### Why create the product type before the product?

Because the product inherits this framework for planning and production behavior.
