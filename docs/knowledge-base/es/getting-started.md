Esta sección ayuda a un usuario nuevo a entender por dónde empezar y en qué orden trabajar dentro de la aplicación.

## Esta sección responde a tres preguntas

- ¿Por dónde empiezo?
- ¿Qué tiene que estar creado antes de lanzar una producción?
- ¿Qué errores conviene evitar desde el principio?

## Idea clave

La aplicación funciona mejor si se sigue el flujo real del trabajo:

1. preparar los datos de referencia,
2. configurar productos y fórmulas,
3. planificar,
4. asegurar compras y stock,
5. asignar lotes reales,
6. ejecutar y controlar.
