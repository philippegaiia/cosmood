Antes de crear su primera producción, compruebe estos puntos.

## Checklist mínima

- el producto existe y está activo,
- el tipo de producto está definido,
- hay una fórmula utilizable,
- los ingredientes críticos están vinculados a proveedores,
- las líneas permitidas tienen sentido,
- sabe si trabajará con un lote aislado o con una oleada.

## Si duda entre lote individual u oleada

- use una producción individual para una necesidad aislada,
- use una oleada si quiere agrupar varios lotes, coordinar compras o seguir una misma carga de trabajo.

## Pregunta típica

### ¿Por qué la aplicación me bloquea más tarde?

Normalmente porque se saltó un prerrequisito al principio: tipo de producto, fórmula, línea permitida o referencia proveedor.
