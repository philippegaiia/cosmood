El calendario da una visión cronológica sencilla de lo que viene.

## ¿Para qué sirve esta página?

Ayuda a:

- detectar rápido los días cargados,
- ver los lotes que vienen,
- revisar la distribución global sin entrar en todo el detalle del board.

## Cuándo usarlo

- para una lectura global rápida,
- para preparar una reunión de planificación,
- para comprobar el impacto de una replanificación.

## Límite importante

El calendario es muy útil para leer, pero el board sigue siendo la vista más precisa para arbitrar capacidad por línea.
