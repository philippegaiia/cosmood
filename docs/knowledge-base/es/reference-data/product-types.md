El tipo de producto es el marco operativo del producto.

## ¿Para qué sirve esta página?

Aquí se define, entre otras cosas:

- el modo de cálculo del lote,
- el tamaño de lote por defecto,
- las unidades esperadas,
- la merma estimada,
- la plantilla de QC compartida,
- las líneas de producción autorizadas,
- la línea por defecto.

## Lo que suele preguntarse el usuario

### ¿Por qué tengo que crear el tipo antes que el producto?

Porque el producto hereda este marco para la planificación y parte del comportamiento productivo.

### ¿Para qué sirven las líneas autorizadas?

Sirven para evitar planificar un tipo de producto en una línea que no corresponde.

## Errores frecuentes

- dejar un tipo demasiado genérico,
- olvidar la línea por defecto,
- usar presets de lote que no corresponden al proceso real.
