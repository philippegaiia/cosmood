El producto es la ficha del SKU o del artículo fabricado.

## ¿Para qué sirve esta página?

Permite relacionar:

- la categoría de producto,
- el tipo de producto,
- el peso neto,
- la fórmula por defecto,
- el packaging,
- la fecha de lanzamiento,
- el estado activo del producto.

## Lo que el usuario suele querer entender

### ¿Por qué aquí es tan importante elegir el tipo de producto?

Porque el producto por sí solo no define la lógica de planificación.

### ¿La fórmula por defecto es obligatoria?

No siempre, pero sin una fórmula útil la producción y el cálculo de necesidades quedan incompletos.

## Errores frecuentes

- crear un producto sin tipo asociado,
- no comprobar la coherencia entre producto y fórmula,
- seguir planificando con productos inactivos.
