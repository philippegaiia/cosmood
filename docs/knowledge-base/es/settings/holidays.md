Los días festivos sirven para hacer más fiables las fechas calculadas y replanificadas.

## ¿Para qué sirve esta página?

Permite definir:

- festivos recurrentes cada año,
- festivos específicos de un año concreto.

## Por qué es importante

Las replanificaciones y los cálculos de fechas pueden saltarse estos días si la opción correspondiente está activada.

## Errores frecuentes

- olvidar un festivo excepcional,
- crear un duplicado entre recurrente y específico,
- pensar que el sistema saltará festivos si aquí no se ha configurado nada.
