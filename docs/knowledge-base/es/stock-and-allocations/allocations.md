Una asignación reserva un lote real para una necesidad real.

## ¿Para qué sirve esta página?

Permite:

- reservar un lote para una producción,
- reservar un lote para una oleada,
- proteger ese stock frente a otros usos,
- mantener una trazabilidad precisa hasta el consumo.

## Lo que suele preguntar el usuario

### ¿Asignar es lo mismo que consumir?

No. La asignación reserva. El consumo llega después, durante la ejecución.

### ¿Por qué la aplicación me dice que la producción no está lista si tengo stock?

Porque a veces ese stock todavía no está asignado en los lotes correctos para arrancar de verdad.

## Errores frecuentes

- olvidar desasignar antes de borrar o reorganizar,
- asignar sin revisar cantidades restantes,
- pensar que una cobertura de compras sustituye a una asignación física.
