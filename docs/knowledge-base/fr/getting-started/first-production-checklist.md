Avant de creer votre premiere production, verifiez les points suivants.

## Checklist minimum

- le produit existe et est actif,
- le type de produit est renseigne,
- une formule exploitable est disponible,
- les ingredients critiques sont references cote fournisseur,
- les lignes autorisees sont coherentes,
- le mode de travail est clair : lot seul ou vague.

## Si vous hesitez entre lot seul et vague

- utilisez une production seule pour un besoin isole,
- utilisez une vague si vous voulez grouper plusieurs lots, suivre une meme charge ou coordonner les achats.

## Question typique

### Pourquoi l'application me bloque plus tard ?

Souvent parce qu'un prerequis a ete saute au debut : type de produit, formule, ligne autorisee, ou referencement fournisseur.
