Le calendrier donne une vue chronologique simple de ce qui arrive.

## Cette page sert a quoi ?

Elle aide a :

- reperer rapidement les jours charges,
- voir les lots a venir,
- verifier la repartition globale sans entrer dans tout le detail du board.

## Quand l'utiliser

- pour une vue globale rapide,
- pour preparer une reunion de planning,
- pour verifier un impact date apres replanification.

## Limite a garder en tete

Le calendrier est tres utile pour lire, mais le board reste la vue la plus precise pour arbitrer la capacite ligne.
