Les referentiels sont les briques de base du systeme.

## Cette section sert a comprendre

- a quoi sert chaque fiche de parametrage,
- ce qui est purement structurel,
- ce qui pilote ensuite les productions, le QC et les achats.

## Reflexe utile

Quand vous vous demandez "pourquoi ce champ existe ?", il faut souvent regarder son effet plus tard dans le flux de planification, d'approvisionnement ou d'execution.
