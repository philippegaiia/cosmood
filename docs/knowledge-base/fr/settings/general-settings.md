Cette page sert a gerer les reglages transverses visibles dans plusieurs documents et ecrans.

## Ce qu'on y configure

- le libelle fournisseur interne,
- le format de date,
- le nom de societe,
- l'adresse societe,
- le numero de TVA.

## Pourquoi c'est important

Ces informations alimentent notamment les documents fournisseur et certains affichages globaux.

## Erreurs frequentes

- oublier que ce sont des parametres visibles a l'exterieur,
- saisir un format de date incoherent avec l'usage equipe,
- laisser des informations societe incompletes sur les bons de commande.
