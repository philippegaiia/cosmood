Les jours feries servent a fiabiliser les dates calculees et replannifiees.

## Cette page sert a quoi ?

Elle permet de definir :

- des jours feries annuels recurrents,
- des jours feries specifiques a une annee.

## Pourquoi c'est important

Les replannifications et calculs de dates peuvent ignorer ces jours si l'option correspondante est active.

## Erreurs frequentes

- oublier un ferie exceptionnel,
- creer un doublon entre recurrent et specifique,
- croire que le systeme saute les jours feries si rien n'est parametre ici.
