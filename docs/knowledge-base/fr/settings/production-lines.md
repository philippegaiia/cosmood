Les lignes de production representent vos voies d'execution physiques.

## Cette page sert a quoi ?

Elle permet de definir :

- les lignes actives,
- leur ordre,
- leur capacite journaliere,
- leur disponibilite pour le planning.

## Question frequente

### Pourquoi cette ligne influence le board ?

Parce que la capacite par jour et par ligne est un des noyaux du planning.

## Erreurs frequentes

- oublier de mettre a jour une capacite devenue obsolete,
- laisser une ligne inactive encore referencee dans les usages terrain,
- ne pas aligner les types de produit avec les lignes reellement autorisees.
